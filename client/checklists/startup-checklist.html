📝 Start up Checklist

More info: KROO SOP

1 ⚡️ Electricity
Check the electricity credit balance and note the amount.
2 🔋 Battery
Verify the battery status and select the appropriate option.
3 🌐 WE Internet / DSL
Check the internet usage and status for each line.
4 📶 Orange Internet / Backup
Enter the current package usage for Orange Internet.
5 📹 DVR / Cameras
Check if the DVR and cameras are operational and reset if needed.
6 ☕️ Unpaid Drinks
Verify if there are any unpaid drinks and notify the responsible person.
7 🔔 Ending Subscriptions
Check for any ending subscriptions and ensure notifications have been sent.
8 📦 Missing Supplies
List any missing supplies in the kitchen or bathrooms.
9 🛠️ General Maintenance
Detail any maintenance required for general facilities.
